# ImpactHub 4.22 进展

本周围绕三层叠加推进：**信息层**铺底、**分析层**出洞察、**传播层**做分发。

## 一、信息层

给画像和榜单打数据地基，形成"启动用的种子 + 可扩展的自动补全"两条路径。

- **种子学者名单**：整理了 87 位 AI/CS 知名学者的结构化候选名单，覆盖 LLM / CV / VLM / Systems / Theory / RL 六个方向，资深 / 中坚 / 新锐三档分布。
- **批量导入**：用户只要有 Semantic Scholar ID，系统就自动完成姓名匹配、GitHub / HF 账号发现、论文 / 仓库 / 模型全量拉取。
- **荣誉补全**：联网搜索为种子学者自动打上 IEEE Fellow / ACM Fellow / 杰青 / 长江 / 院士等标签，人工复核后入库。
- **履历抓取**：自动整理本硕博 + 任职履历（含导师、机构、任期），前端按时间倒序分「教育背景 / 职业经历」两栏呈现。
- **引用者画像**：对高 h-index 引用者识别荣誉，表达 "谁在引用你" 而不仅是 "有多少人引"。

**待补**：种子名单规模偏小、方向以理论 AI 为主，下周扩到 200-500 人；GitHub 账号姓名模糊匹配的错配率需要降下来。

## 二、分析层

从单一指标或排名，升级到"研究者的立体画像"，全部由 LLM 综合多维数据生成，避免硬编码规则带来的边界错判。

- **研究演化树**：把人的研究路径画成树——主干是时间线，分支是不同研究方向/职业阶段，体现方向转移和方法演化，而不是静态档案。
- **能力画像（多方向立体）**：识别研究者真正从事的 1-4 个方向，在每个方向单独给角色（开创者 / 早期采用者 / 扩展者 / 跟随者）+ 占比 + 成就概括 + 代表作。一个人可以"在 A 方向是开创者，在 B 方向是扩展者"，比单一标签更立体。
- **研究者人格（12 种 meme 画像）**：仿 MBTI 做了 12 种带梗的人格（老神仙 GOAT / 组里老大 PI / 独狼 WOLF / 卷王 JUAN / 论文工厂 KPI 等），每种有自嘲 tagline 和插画。
- **年度诗篇**：仿小红书"年度诗篇"结构，从当年论文 / 引用增长 / 代表作 / 仓库里提炼 10-14 行数据驱动的现代短诗，代表作过滤第一作者 / 通讯位置，避免把挂名合作写成"代表作"。
- **排行榜**：总榜 + 年轻学者榜（首篇 <10 年） + 六个方向榜，前 10000 名精确排位，之后只显示百分位，兼顾可比性与审慎性。

**待补**：还没做"轨迹相似度"——和种子学者对照找出"你最像谁"（例如"你的 2015-2020 像 Yoshua Bengio 1988-1993"）；peer 之间的相似度匹配也未做。

## 三、传播层

让画像能被分享、被扫读、被记住。

- **分享卡片系统**：三种可切换的卡片类型
  - **影响力卡**：综合雷达 + 核心指标 + 趣味标签
  - **人格卡**：AI 生成的 meme 风格插画 + 名字 + 维度条 + tagline
  - **年度诗篇卡**：深色渐变 + 居中诗句 + 数据印章
- **要素自定义**：影响力卡支持 6 种展示要素的勾选（社区讨论 / 引用 / 平台链接 / CCF / 开源数据 / AI 标签），未勾选的内容会从卡片和雷达图中同步隐藏（替换为中等位置值），用户可以只分享想展示的部分。
- **AI 插画**：用 AI 一次性生成了 12 张低多边形 flat design 风格的人格插画，带黑色幽默感（卷王的眼袋+巨型咖啡杯 / 论文工厂的机械式流水线 / 苦行僧的空洞眼神），风格统一、可直接导出分享。
- **页面重构**：把原来 9 个 tab 压缩成 4 个顶级分组（经历 / 成果 / 影响 / 导出），"经历"置前默认打开，形成"先识人、再看产出、最后看影响"的浏览动线。
- **文档站**：新增 `/docs` 页面展示系统总览，带层级目录 + 滚动高亮，所有 12 种人格的插画也在文档里做了画廊展示。

**待补**：卡片目前是整张导出，没支持"只分享某一个子要素"（例如只分享演化树或只分享年度诗篇中的一句）；分享卡的视觉还偏功能性，可以再做一版更具网感、更适合在社交媒体流里抢眼球的版本。

## 下周优先级

1. **轨迹相似度匹配**——最能传播的"对照"能力
2. **原子化分享**——每个画像支持单独拆出来分享
3. **共同作者网络**——从论文作者列表构建可视化
4. **Peer 相似度**——全库学者相似度近邻查询
